# TravelSystem
数据结构课程设计
题目：景区旅游信息管理系统
功能：
      创建景区景点分布图；
      输出景区景点分布图（邻接矩阵）
      输出导游线路图（深度优先策略）
      判断导游线路图有无回路（拓扑排序，查找入读大于1的景点）
      求两个景点间的最短路径和最短距离（弗洛伊德算法）
      输出道路修建规划图（prime算法）
      主程序用菜单选项供用户选择功能模块
